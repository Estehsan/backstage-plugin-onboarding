# Screenshots

Place the following screenshot files in this directory:

1. **onboarding-dashboard.png** - The main onboarding page showing phase-based task lists (Day 1, Week 1, Week 2, Month 1) with the progress bar at the top.

2. **task-detail-panel.png** - An expanded task showing the detail panel with description, resources/links, assignee info, and status controls (mark done/blocked).

3. **team-overview.png** - The team/manager view showing a table of active joiners with their completion percentages, blocked task counts, and start dates.

## Recommended dimensions

- Width: 1200-1400px
- Format: PNG (preferred) or GIF for animated walkthroughs
- Capture at 2x resolution for retina displays if possible
